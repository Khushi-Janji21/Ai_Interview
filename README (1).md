# 🤖 AI Interview — Powered by Antigravity

> **Practice smarter. Interview better. Land your dream job.**

An AI-powered interview preparation platform that helps anyone — students or professionals — ace their interviews with real-time AI feedback, session tracking, and insightful reports.

---

## ✨ Features

- 🧠 **AI Interview Questions & Feedback** — Get dynamic interview questions generated by OpenAI and receive instant, intelligent feedback on your answers
- 📊 **Session History & Reports** — Track your progress over time with detailed session history and performance reports
- 🛡️ **Admin Dashboard** — Manage users, sessions, and platform analytics from a powerful admin panel
- 🔐 **Authentication** — Secure login and registration system
- 👤 **User Profiles** — Personalized profiles to track your interview journey

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React.js + Vite |
| Backend | Node.js + Express.js |
| AI Engine | OpenAI API |
| Styling | CSS Modules |
| Charts | Recharts |

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm
- OpenAI API Key

### Installation

```bash
# Clone the repository
git clone https://github.com/Khushi-Janji21/Ai_Interview.git

# Navigate to the project
cd Ai_Interview

# Install frontend dependencies
cd client
npm install

# Start the frontend
npm run dev
```

### Environment Variables

Create a `.env` file in your backend folder:

```env
OPENAI_API_KEY=your_openai_api_key
PORT=5000
```
